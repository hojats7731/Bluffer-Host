# Host audio assets

Add royalty-free OGG files (optional):

| File | Use |
|------|-----|
| `background.ogg` | Looping lobby/game music |
| `reveal_sting.ogg` | Short sting when each lie is revealed |
| `join_chime.ogg` | Player joins lobby |

The app runs without these files; `AudioManager` skips missing assets.
